Эта страница в основном такая же, как и обычная страница комбинаций поплавков, за исключением того, что вы можете загружать данные из opskins.

Для этого вам нужно использовать свой собственный ключ api

Чтобы получить свой, перейдите в настройки BitSkins. Вам нужно будет включить 2fa, чтобы получить собственный ключ API.

Введите параметры поиска и нажмите кнопку «Добавить», чтобы добавить данные в раздел ниже.

Чтобы найти минимальное и максимальное значения float для скина, до которого вы хотите обменять, ознакомьтесь с https://csgostash.com/

Вы можете добавить более одного поиска и выбрать, сколько чисел с плавающей запятой из каждого должно использоваться для генерации комбинаций.

Если вы хотите, чтобы один поиск давал больше результатов, чем другой, просто измените параметры на «x, x, x, x», чтобы загружать x чисел с плавающим флоатом из каждого поиска (search0 - searchchn)

Я не пытаюсь украсть ваш ключ api, но поскольку существует много мошенников, а код в настоящее время не является открытым исходным кодом, вы можете просто ввести свои спецификации, и на странице будет ссылка, по которой вы можете ввести свой собственный ключ API

Как только у вас будет открыта вкладка с данными, скопируйте + вставьте ее в текстовое поле ниже «Всего возможных комбинаций» (убедитесь, что между наборами данных есть полная строка).

Не забудьте дважды проверить любые комбинации с https://csgo.exchanget/. Таким образом, если набор из 10 скинов немного отличается, вы можете продать их до того, как не сделаете обмен.
